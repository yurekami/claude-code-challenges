# Universal Interface Solution

## Task Integration Examples

### Morning Routine in One Session
```
1. "Review my git commits from yesterday"
2. "Write a status update for my team"
3. "Plan today's work based on open issues"
4. "Generate a quick script to backup my config files"
5. "Help me respond to this technical email professionally"
```

### Cross-Domain Problem
```
"I need to:
1. Fetch data from an API (code)
2. Analyze it statistically (data science)
3. Create visualizations (design)
4. Write a report explaining findings (writing)
5. Present it in a slide deck (presentation)

Help me with each step."
```

### Workflow Automation Example
```bash
#!/bin/bash
# daily-workflow.sh - Combines multiple tasks

# Update repos
claude "Generate commands to update all git repos in ~/projects"

# Process emails
claude "Draft responses to these meeting requests: [list]"

# Plan day
claude "Based on my calendar and backlog, suggest today's priorities"

# Generate report
claude "Create weekly progress report from my commits this week"
```

## Success Indicators

- Handle 10+ different task types smoothly
- Build effective automated workflows
- Maintain productivity across domains
